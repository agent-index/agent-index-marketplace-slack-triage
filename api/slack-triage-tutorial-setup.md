# Setup

This skill requires no setup. The tutorial is available immediately after installation.

When you invoke `@ai:slack-triage-tutorial`, Claude will either walk you through a guided tour of how slack-triage works or answer a specific question you have about the system.

The tutorial reads your existing setup (if you have one) to provide relevant examples, but it doesn't create or modify any configuration.
